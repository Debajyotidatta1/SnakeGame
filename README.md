# SnakeGame

A simple browser-based snake game where the snake hunts a small mouse across a matrix-style grid.

## Features

- Matrix-like box grid layout
- Snake movement with growing tail
- Mouse target that respawns after each catch
- Score and best score tracking
- Pause and restart controls
- Keyboard and touch controls

## How to Run

1. Open `index.html` in any modern browser.
2. Start moving the snake with the keyboard or touch controls.

## Controls

- `Arrow Keys`: Move the snake
- `W`, `A`, `S`, `D`: Move the snake
- `Space`: Pause or resume

## Files

- `index.html` - Game structure
- `styles.css` - Visual design and layout
- `script.js` - Game logic

## Game Goal

Catch as many mice as you can without hitting the wall or the snake's own trail.
